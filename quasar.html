Instalace

yarn global add @quasar/cli

Vytvoření projektu

  yarn create quasar
  # OR
  npm init quasar

Konfigurace

quasar.conf.js

Konfigurační soubor, který nastavuje veškerou funkcionalitu, kterou poskytuje Quasar:

  • Výběr použitých komponent, pluginů a directives.
  • Jazyk/jazyky aplikace
  • Knihovna/y ikon
  • Defaultní knihovna pro ikony
  • Nastavení vývojového serveru:
    • port
    • HTTPS mode
    • hostname
  • Nastavení CSS animací
  • Nastavení bootování a jeho pořadí ve spouštění ze složky /src/boot.
  • Zahrnutí CSS dokumentů, které maji být zahrnuty.
  • Nastavení PWA aplikace
  • Nastavení Electronu pro generování desktopové aplikace

Základní koncepty

Quasar využívá Single file component z Vue, ale kromě toho Vue rozšiřuje o následující nástroje:

  • Quasar directives:
<div v-ripple>Click Me</div>
  • Quasar components
<div>
    <q-btn @click="doSomething" label="Do something"/>
    <q-icon name="alarm"/>
</div>
  • Quasar plugins musí se uvést v quasar.conf.js
framework: {
    plugins: ['Notify', 'BottomSheet']
}
}

<koponenta.vue>

<template>
    <div>
        <q-btn
                @click="$q.notify('My message')"
                color="primary"
                label="Show a notification"
        />

        <q-btn
                @click="showNotification"
                color="primary"
                label="Show another notification"
        />
    </div>
</template>

<script>
    import {useQuasar} from 'quasar'

    export default {
        setup() {
            const $q = useQuasar()

            function showNotification() {
                $q.notify('Some other message')
            }

            return {
                showNotification
            }
        }
    }
</script>

Styl a grafika

Písmo

Přidání vlastního písma:

  1. Nakopíruj písmo s koncovkou .woff do adresáře ./src/css/fonts/[customfont.woff].
  2. Deklaruj tvoje nové písmo v ./src/css/app.{css|sass|scss|styl}:
@font-face {
  font-family: customfont;
  src: url(./fonts/customfont.woff);
}

// declare a class which applies it
.my-font {
  font-family: 'customfont';
}

Barva

Quasar má vlastní předdefinované barvy v souboru /src/css/quasar.variables.sass, které můžeme libovolně měnit. Vlastní barevný styl lze nastavit v Theme Builder. Quasar má vlastní color list: Color Palette | Quasar Framework

Použití:

  • barva písma:
<!-- changing text color -->

<p class="text-primary">....</p>
  • barva pozadí:
<!-- changing background color -->
<p class="bg-positive">...</p>

Dynamická (programatická) změna barvy: Color Palette | Quasar Framework

Mezery a okraje

Quasar používá variabilní nazvy tříd, které nastavují padding a margin

Syntaxe je následující:

q-[p|m][t|r|b|l|a|x|y]-[none|auto|xs|sm|md|lg|xl]
  • třída začíná vždy písmenem q
  • poté se nastavuje typ: p - padding nebo m - margin
  • následuje směr (direction):
    • t - top
    • r - right
    • b - bottom
    • l - left
    • a - all
    • x - left and right
    • y - top and bottom
  • nakonec je velikost:
    • xs - extra malé
    • sm - malé
    • md - střední
    • lg - velké
    • xl - extra velké

Pokud nechceme margin a/nebo padding, tak použijeme tyto classy:

  • no-margin
  • no-padding

Příklad: SpacingPage.vue

<template>
    <q-page>
        <div class="q-ma-xl">
            <p>Ahoj</p>
        </div>
        <div class="q-pa-xl">
            <p>Ahoj</p>
        </div>
        <div class="no-padding">
            <p>Ahoj</p>
        </div>
    </q-page>
</template>

<script>
    export default {
        name: 'SpacingPage',
    }
</script>

<style scoped>
    div {
        border: 5px solid red;
    }
</style>

routes.js

const routes = [
    {
        path: '/',
        component: () => import('layouts/MainLayout.vue'),
        children: [
            {path: '', component: () => import('pages/IndexPage.vue')},
            {path: 'spacing', component: () => import('pages/SpacingPage.vue')}
        ]
    },

    // Always leave this as last one,
    // but you can also remove it
    {
        path: '/:catchAll(.*)*',
        component: () => import('pages/ErrorNotFound.vue')
    }
]

export default routes

Komponenty

Quasar knihovna obsahuje nesčetně UI komponent, které můžete použít a postavit si svojí webovou stránku/aplikaci jako LEGO. Každá komponenta má svojí stránku, kde je mnoho příkladů a také API dokumentace. Příklad si ukážeme na jednoduchém tlačítku a input boxu. Do input boxu napíšeme text a po stisknutí tlačítka se nám přepíše na velká písmena. Nebude nás až tolik zajímat logika jako stylování.

ComponentsPage.vue

<template>
    <q-page class="q-pa-xl">
        <div class="row justify-center items-end">
            <div class="q-pa-sm col-3">
                <q-input v-model="text" label="Zadej text..."/>
            </div>
            <div class="q-pa-sm col-1" @click="submit">
                <q-btn :loading="submitting" label="Submit" glossy/>
            </div>
        </div>
        <div class="row justify-center">
            <q-card class="my-card">
                <q-card-section>
                    {{ cardText }}
                </q-card-section>
            </q-card>
        </div>
    </q-page>
</template>


<script>
    import {ref} from "vue";

    export default {
        name: "ComponentPage",
        setup() {
            let text = ref('')
            let cardText = ref('')

            const submitting = ref(false)

            function submit() {
                submitting.value = true
                setTimeout(() => {
                    cardText.value = text.value
                    submitting.value = false
                }, 3000)
            }

            return {
                text,
                cardText,
                submitting,
                submit
            }
        },
    }
</script>
